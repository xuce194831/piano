# piano
`一个的电子钢琴游戏`

### 功能
* 按下琴键有反馈，琴键有被按下和弹起效果
* 按下琴键发出相应的琴音
* 手指可连续划屏，发出连续琴音

### 硬件平台：
* GEC-6818
* ARM-Cortex A53

### 软件平台：
* linux-3.4.39
