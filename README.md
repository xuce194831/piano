# piano
an application played on ARM-A53-based platform

一个基于粤嵌 GEC6818 开发板（搭载了ARM Cortex-A53八核处理器）的电子钢琴案例

功能：

1 按下琴键有反馈，琴键有被按下和弹起效果

2 按下琴键发出相应的琴音

3 手指可连续划屏，发出连续琴音
